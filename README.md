# UX Estimate Calculator

A client-side calculator for planning UX project scope, timeline, and budget.

**Live demo:** [andreasjohanssonux.github.io/ux-estimate-calculator](https://andreasjohanssonux.github.io/ux-estimate-calculator/)

## Features

- **Smart Templates** - Quick-start presets for common project types (Mobile App MVP, Website Redesign, Design System, Research Sprint, Usability Audit)
- **Industry Benchmarks** - Typical hour ranges displayed alongside each activity field
- **Dynamic Suggestions** - Real-time tips and warnings based on your inputs
- **Collapsible Sections** - Clean up the interface by collapsing completed sections
- **Export for AI Review** - Copy a formatted summary to paste into ChatGPT/Claude for feedback
- **Print-friendly** - Clean print stylesheet for client proposals

## Project Types Covered

### Research Phase
- Research planning
- Stakeholder interviews
- User interviews
- Personas
- User journey maps

### Design Phase
- UX vision
- Information architecture
- Design concepts & revisions
- Usability testing

### Additional
- Documentation
- Meetings & communication
- Buffer/padding percentage

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- No dependencies or build step
- Works entirely client-side

## Local Development

Just open `index.html` in a browser. No server required.
